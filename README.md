# FreeRTOS 기반 자율주행 RC카

> STM32F411RE와 다중 센서를 활용해 장애물 회피 자율주행과 Bluetooth 수동 제어를 구현한 소형 모빌리티 시스템입니다.

## 프로젝트 정보

- 개발 기간: 2026.06.08 ~ 2026.06.17
- 개발 형태: 개인 프로젝트

## 프로젝트 개요

FreeRTOS 기반의 3-Task 구조로 센서 측정, 자율주행 판단, 수동 제어 기능을 분리했습니다. 3방향 초음파 센서로 장애물을 감지하고 좌·우 거리 차이에 따라 모터 속도를 보정하며, Bluetooth 명령을 통해 자율주행과 수동주행 모드를 전환할 수 있습니다.

## 주요 기능

- 3방향 초음파 센서 기반 장애물 감지 및 회피
- 좌·우 거리 오차를 이용한 P 제어 방식의 차동 조향
- Bluetooth 앱을 통한 자율·수동 모드 전환 및 차량 제어
- IR 센서를 이용한 승객 탑승 감지
- LED·부저·DHT11을 활용한 차량 상태 알림

## Task 구성

| Task | 역할 |
|---|---|
| `SensorTask` | 좌·전방·우측 초음파 거리 측정 및 IR 센서 상태 확인 |
| `DriveTask` | 장애물 회피 판단 및 차동 구동 기반 자율주행 제어 |
| `ManualTask` | Bluetooth 명령 처리 및 자율·수동 모드 전환 |

센서 데이터는 Mutex로 보호하여 `SensorTask`와 `DriveTask` 사이에서 안전하게 공유합니다.

## 기술 스택

`STM32F411RE` · `FreeRTOS` · `C` · `STM32 HAL` · `UART` · `PWM` · `GPIO` · `Bluetooth` · `DHT11`

> 참고: DHT11 측정 로직은 구현되어 있으나, 현재 Bluetooth 전송값은 Task 간 타이밍 문제로 고정값을 사용합니다.
